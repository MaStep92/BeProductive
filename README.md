### Manual setting:

#### Installation

1 - Clone project
```
$ mkdir BeProductive
$ cd BeProductive
$ git clone https://github.com/MaStep92/BeProductive.git
```

2 - Install dependencies
```
$ cd BeProductive
$ npm i
```

3 - Run dev server
```
$ cd ../BeProductive
$ npm run start
```
